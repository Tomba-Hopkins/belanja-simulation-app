## Setup with Docker :

1. docker pull 

``` bash
docker pull hopkinserstomba/mylab-nasgor-edition
```

2. git clone

``` bash
git clone https://github.com/Tomba-Hopkins/my-ctf-lab1.git
```


3. cd dir

``` bash
cd my-ctf-lab1
```

4. docker-compose

``` bash
docker-compose up
```

5. open browser

